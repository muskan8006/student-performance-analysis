# 🎯 Student Performance Analysis 

## 📌 Objective

## 📊 Features
- Data Cleaning & Preprocessing
- Feature Engineering
- Data Visualization
- Machine Learning Prediction
- Streamlit Web App

## ▶️ How to Run
pip install -r requirements.txt
